## Описание проекта

Данный проект создан в рамках лабораторной работы по изучению CSS-фреймворка Bootstrap 5. Работа включает в себя создание многостраничного сайта-портфолио с использованием различных компонентов Bootstrap.

## Что было изучено:

- Подключение Bootstrap через CDN
- Система сеток (Grid System) для создания адаптивных макетов
- Компоненты: Navbar, Cards, Forms, Modal
- Утилитные классы Bootstrap
- Адаптивные изображения
- Создание многостраничных проектов
- Кастомизация стилей через собственный CSS-файл

1. Скачайте или клонируйте репозиторий
